My CSS Journey

Cascading style sheets, or CSS. Simply put, CSS is a design language that enhances a website's visual appeal beyond that of plain or uninteresting text. The visual structure, layout, and aesthetics are determined by CSS, whereas the text content is mostly determined by HTML. HTML is a markup language, whereas CSS is a language for creating style sheets. When considering CSS, keep "look and feel" in mind.

Differnces between paddings, margins, and borders

Margin Padding Border
The outer space of an element i.e. margin is the space outside the border. The inner space of an element i.e.padding is space inside the elements border. A border that goes around the padding and content.
It can be negative or any float number. It does not allow negative values. Border
We can set the margin to auto. We cannot set the padding to auto. Border
Styling of an element such as background color does not affect the margin. Padding is affected by the styling of an element, such as background color. Border
